## Usage
```
$ webcli -h

Usage: index [options] <name>


  Options:

    -V, --version     output the version number
    -t --type [type]  vue-eslint default is vue-eslint
    -q --quiet        will not print anything
    -h, --help        output usage information
```

### Example
- `webcli -t vue-eslint demo`
